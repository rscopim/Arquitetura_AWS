        Gateway de API da Amazon
- O Amazon API Gateway é um serviço totalmente gerenciado que facilita aos desenvolvedores criar, publicar, manter, monitorar e proteger APIs em qualquer escala. 

- As APIs atuam como porta de entrada para que os aplicativos acessem dados, lógica de negócios ou funcionalidades de seus serviços de back-end. 

- O API Gateway lida com todas as tarefas envolvidas na aceitação e processamento de centenas de milhares de chamadas de API simultâneas, incluindo gerenciamento de tráfego, suporte a CORS, autorização e controle de acesso, otimização, monitoramento e gerenciamento de versão de API. 

- Usando o API Gateway, você pode criar APIs RESTful e APIs WebSocket que permitem aplicativos de comunicação bidirecional em tempo real. O API Gateway oferece suporte a cargas de trabalho em contêineres e sem servidor, bem como a aplicativos da web. 

- O AWS Lambda permite executar código sem provisionar ou gerenciar servidores. Você paga apenas pelo tempo de computação consumido.

- Com o Lambda, você pode executar código para praticamente qualquer tipo de aplicativo ou serviço de back-end, tudo sem administração. Basta fazer upload do seu código e o Lambda cuidará de tudo o que é necessário para executar e dimensionar seu código com alta disponibilidade. Você pode configurar seu código para ser acionado automaticamente a partir de outros serviços da AWS ou chamá-lo diretamente de qualquer aplicativo web ou móvel.

    Você sabe?
- API Gateway é um serviço AWS totalmente gerenciado que permite aos desenvolvedores criar, publicar e gerenciar APIs para seus aplicativos. Ele atua como uma porta de entrada para seus serviços de back-end, permitindo que você exponha com segurança suas APIs aos clientes, lide com autenticação e autorização e aplique diversas transformações e otimizações.

- AWS Lambda é um serviço de computação sem servidor fornecido pela Amazon Web Services. Ele permite que os desenvolvedores escrevam e executem código sem provisionar ou gerenciar servidores. Uma função Lambda é um trecho de código que executa uma tarefa específica ou trata de uma solicitação de API específica. Pode ser escrito em várias linguagens de programação e é acionado por eventos, como uma solicitação de API.

- A integração do Lambda no API Gateway refere-se ao processo de conexão de um endpoint de API com uma função Lambda. Ele permite que o API Gateway invoque a função Lambda em resposta às solicitações de API recebidas, permitindo que a função processe a solicitação, execute a lógica de negócios e gere uma resposta.
